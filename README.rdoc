= Fluentd event to Graphite queue plugin like StatD.

= Installation

  $ fluent-gem install fluent-plugin-graphite

= Usage

    <match graphite>
      type graphite
    </match>

    fluent_logger.post('graphite', {
      :key => 'mykeyname',
      :type => 'count',
      :value => 1
    })
